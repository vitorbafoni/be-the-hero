## Features

Essa aplicação foi desenvolvida com o intuito de praticar Node.js e React.js.

- 💹 **Node Js** — Utilizado para criar o backend da API.
- ⚛️ **React Js** — Utilizado para criar a interface do projeto.


## Getting started

1º Baixe ou clone o repositório: https://github.com/vitorbafoni/be-the-hero.git e abra o arquivo index.html<br>
2º Acesse a pasta "frontend" pelo terminal e digite "npm start".<br>
3º Acesse a pasta "backend" abrindo outro terminal e digite "npm start".
